# 변수의 비밀 생명주기

## 변수 사용 가능 시점 🎬

greeting()는 선언하기전에 실행 했는데 왜 작동할까?

```
greeting();

function greeting(){
  console.log("안녕하세요");
}
```

함수 호이스팅 때문이다.

### 호이스팅 🚀

> 선언은 스코프 아래에 있더라도 스코프 시작 부분에서 변수의 가시성이 확보되는 걸 뜻한다.

---

📌 함수 선언문으로 함수를 선언하면 함수 이름에 해당하는 식별자가 스코프 최상단에 등록되고 함수 참조로 그 값이 자동으로 초기화된다.
